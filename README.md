# Arma-Reforger-HOTAS-Mapper
To create your own bindings, Arma Reforger requires a corresponding configuration file.
<T16000 Bindings.conf> is such a file. 
It must be saved in the following location:
C:\Users\YOURNAME\Documents\My Games\ArmaReforger\profile\.save\settings\customInputConfigs
If the <settings\customInputConfigs> folder is not present in the .save folder, as was the case for me, simply create it and place the file in <customInputConfigs>.

Now you can start the tool. At the bottom, there is a button to load a .conf file. Click this button and select the file mentioned above. All available settings should be displayed. If you select a line and enter a value, it will be updated and saved immediately.
When you are finished, you can simply close the program.
